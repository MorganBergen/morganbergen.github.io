# copilot setup

`:Copilot status` will show you the current status of the copilot.

`:Copilot enable` 

`:Copilot start`

**pwd to enable and write for configs**

`~/config/nvim/init.vim`

**write out mapping on `init.vim`**
```
let g:copilot_filetypes = {'markdown: v:true}
```
